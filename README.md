# Random Password Generator


This project was created to help the user generate a strong random password. This application emphasizes the use of Javascript to produce passwords based on user inputs and answers.

# Motivation

The user would like to be able to generate a new password every time they need one. The password has to be a mix of different combinations which can include lower case characters, uppercase characters, special characters, symbols.
