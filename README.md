## 03 Javascript / Password Generator

## Demo Link
https://chanraksarwat.github.io/JavaScript-Password-Generator/

## Tasks
  This project was designed as a secure randome password generator as a homework.
  This application emphasizes the use of Javascript to generate a random, secure password for the user.

## User Story
  User should be able to click on the Generate Password button and present with multiple alerts. After user has responded and confirmed to those alerts only then will the program display random password inside of the text box.
